SeleniumTest
Script em Python 3 de template básico para utilizar o Selenium.

<h1>Instalação e Requisitos</h1>

<h2>Instale o selenium no seu Python.</h2>
pip install selenium
<h2>Baixe os drivers para os navegadores</h2>
Baixe e copie os drivers que vc deseja utilizar para uma pasta e inclua no path do windows.
